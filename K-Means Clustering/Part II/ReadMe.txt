The code uses python 2.7 version SDK and interpreter. 
-------------------------------------------------------------------
The zip file submitted contains:
tweets-k-means.py (code file)
InitialSeeds.txt (input initialSeedsFile)
Tweets.json	(input TweetsDataFile)
Report_PartII.txt
tweets-k-means-output.txt
Experiment results folder (contains the results for the 5 experiments conducted)

-------------------------------------------------------------------
To Compile and run the program:

1. Open the terminal and set the directory to the one containing the python code and input files.
2. Type the following command on the comman line to execute the python code:
python tweets-k-means.py <No. Of Clusters> <initialSeedsFile> <TweetsDataFile> <outputFile>

Example command:
python tweets-k-means.py 25 InitialSeeds.txt Tweets.json tweets-k-means-output.txt
